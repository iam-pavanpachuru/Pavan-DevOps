What is Git? What is GitHub Actions?

What is Git?

Git is a version control system. It tracks changes in your code so you can go back, compare versions, and work with a team safely.

  • Undo mistakes using history
  • Work in branches without breaking main code
  • See who changed what & when

What is GitHub Actions?

GitHub Actions is an automation tool inside GitHub. It can build, test, and deploy your project automatically.

In this demo, GitHub Actions takes this static website and deploys it to GitHub Pages every time I push to the main branch.
